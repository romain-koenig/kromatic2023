À propos de la conception de ce site

Ce site web a été concu selon les principes d'écoconception, et avec les bonnes pratiques en termes d'accessibilité.

Écoconception

Ce site est conçu avec les principes d'écoconception, afin de réduire son impact environnemental. Il est hébergé chez un hébergeur local et responsable. Le site est statique, et ne nécessite pas de base de données. Il est conçu avec le générateur de site statique Eleventy, qui est très rapide et efficace. Les images sont compressées et optimisées pour le web.

Accessibilité

Ce site est conçu avec les bonnes pratiques en termes d'accessibilité. Il est conforme aux normes WCAG 2.1 AA. Il est testé avec le validateur WAVE.

Tests et validation

Tests Lightouse

Ce site est testé avec Lighthouse, afin de garantir de bonnes performances, une bonne accessibilité, et de bonnes pratiques et un bon référencement.

Écoindex

Ce site est testé avec l'outil Écoindex, afin de garantir de bonnes performances en termes d'impact environnemental.

Carbon Badge

Ce site est testé avec le badge Carbon, afin de garantir un site à faible impact environnemental.

Validation W3C

Ce site est testé avec le validateur W3C, afin de garantir un code HTML valide et donc accessible.

WAVE

Ce site est testé avec le validateur WAVE, afin de garantir un site accessible (Visibilité, lisibilité, navigation au clavier, etc.)

Cookies et tracking

Ce site n'utilise pas de cookies, et ne fait pas de tracking. Il n'y a pas de boutons de partage vers les réseaux sociaux, afin de garantir votre vie privée.
